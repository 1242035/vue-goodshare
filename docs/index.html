Requirements

  • Vue.js 2.x (github.com/vuejs/vue)

Install component

$ npm i vue-goodshare --save

Testing

Clone repository from GitHub:

$ git clone https://github.com/koddr/vue-goodshare.git

Go to example folder:

$ cd vue-goodshare/example

Start simple Python 3.x HTTP server (only macOS and Linux):

$ python3 -m http.server 4000 --bind localhost
